# Developer Brief — The Legacy Circle Mobile App

## Project Overview
The Legacy Circle is an AI-powered educational app for children ages 2–14. The web app is fully built on Base44. We need a developer to wrap it as a native mobile app and submit it to the Apple App Store and Google Play Store.

## What's Already Done
- ✅ Full web app built and published (Base44)
- ✅ GitHub repo with all code (github.com/ddortese-pixel/Legacy-Circle)
- ✅ App icon (1024x1024px PNG)
- ✅ Splash screen
- ✅ Privacy policy (COPPA compliant, live URL)
- ✅ App store listing (name, description, keywords, screenshots)
- ✅ Google Play feature graphic

## What We Need
1. **Capacitor setup** — Wrap the existing Base44 web app using Ionic Capacitor to generate native iOS and Android builds
2. **iOS build** — Compile and submit to Apple App Store via Xcode
3. **Android build** — Compile and submit to Google Play Console
4. **TestFlight setup** — Set up internal beta testing on Apple TestFlight
5. **Store submission** — Handle full submission process for both stores including metadata upload

## Technical Details
- Web app framework: React (Base44 platform)
- GitHub repo: github.com/ddortese-pixel/Legacy-Circle
- Target platforms: iOS 15+ and Android 10+
- Age rating: 4+ (Apple) / Everyone (Google Play)
- COPPA compliant — child privacy handled

## Assets Provided
- App icon: 1024x1024px PNG
- Screenshots: 5 screens (iPhone 6.7" format)
- Feature graphic: 1024x500px (Google Play)
- Privacy policy URL
- Full app store listing copy

## Developer Accounts
- Apple Developer account: needs to be set up ($99/year) OR developer can use theirs and transfer
- Google Play Developer account: needs to be set up ($25 one-time) OR developer can use theirs and transfer

## Timeline
Looking to submit to both stores within 4–6 weeks.
